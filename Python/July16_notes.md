# 🐍 Python Basics 

## 1. Python
Python is a high-level, interpreted, general-purpose programming language that executes code line by line.

---

## 2. Procedural Execution
Procedural execution means the program runs instructions sequentially from top to bottom.

---

## 3. Libraries
A library is a collection of pre-written code that provides reusable functions and tools.

---

## 4. Variables
A variable is a name used to store data in memory, created using assignment.

---

## 5. Dynamic Typing
Dynamic typing means Python automatically determines the data type of a variable at runtime.

---

## 6. Data Types
Data types define the type of value a variable can store.

---

## 7. Integer (int)
An integer is a whole number without a decimal.

---

## 8. Float
A float is a number that contains decimal points.

---

## 9. Boolean
A boolean is a data type that represents either True or False.

---

## 10. String
A string is a sequence of characters enclosed in quotes.

---

## 11. Complex Data Type
A complex number consists of a real part and an imaginary part.

---

## 12. Type Casting
Type casting is the process of converting one data type into another.

---

## 13. Implicit Casting
Implicit casting is automatic type conversion performed by Python.

---

## 14. Explicit Casting
Explicit casting is manual type conversion performed by the programmer.

---

## 15. Variable Naming Rules
Variable naming rules define how variables should be named, such as not starting with numbers or special characters and avoiding keywords.

---

## 16. Naming Convention
A naming convention is a standard way of naming variables, commonly using snake_case in Python.

---

## 17. Operators
Operators are symbols used to perform operations on variables and values.

---

## 18. Arithmetic Operators
Arithmetic operators are used to perform mathematical calculations.

---

## 19. Comparison Operators
Comparison operators are used to compare two values and return a boolean result.

---

## 20. Logical Operators
Logical operators are used to combine multiple conditions.

---

## 21. Assignment Operators
Assignment operators are used to assign and update values in variables.

---

## 22. Input Function
The input() function is used to take user input during program execution.

---

## 23. Print Function
The print() function is used to display output on the screen.

---

## 24. F-String
An f-string is a way to format strings by embedding variables inside them.

---

## 25. Dot Format Method
The format() method is used to insert values into a string using placeholders.

---

## 26. Boolean Expressions
A boolean expression is an expression that evaluates to either True or False.

---

## 27. Branching
Branching is the process of making decisions in a program using conditions.

---

## 28. if Statement
An if statement executes a block of code only when a condition is true.

---

## 29. if-else Statement
An if-else statement executes one block of code if the condition is true and another if it is false.

---

## 30. if-elif-else Statement
An if-elif-else statement is used to check multiple conditions in sequence.

---

## 31. Nested if
A nested if is an if statement inside another if statement.
